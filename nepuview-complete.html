PlayerFragment.kt — Orientation handlingoverride fun onConfigurationChanged(newConfig: Configuration) {
    super.onConfigurationChanged(newConfig)
    if (newConfig.orientation == Configuration.ORIENTATION_LANDSCAPE) {
        enterFullscreen()   // StatusBar + NavBar ausblenden
    } else {
        exitFullscreen()    // wieder einblenden
    }
}

private fun enterFullscreen() {
    requireActivity().window.decorView.systemUiVisibility = (
        View.SYSTEM_UI_FLAG_FULLSCREEN
        or View.SYSTEM_UI_FLAG_HIDE_NAVIGATION
        or View.SYSTEM_UI_FLAG_IMMERSIVE_STICKY
    )
}
02 — Download System

HLS-Downloads & Offline-Playback

Wie nepu.to Videos ausliefert
HLS (HTTP Live Streaming) — M3U8 Playlists

nepu.to streamt über HLS. Das bedeutet: kein einzelnes Video-File, sondern eine .m3u8 Playlist + hunderte kleine .ts Segmente. Zum Herunterladen muss die M3U8-URL zuerst aus dem WebView abgefangen werden.

1
M3U8-URL abfangen (shouldInterceptRequest)
Der Player-WebView lädt die Film-Seite. Sobald der HLS-Stream startet, erscheint ein Request auf eine .m3u8 URL. Diese URL wird abgefangen und in der ViewModel gespeichert.
2
Download-Button erscheint im Player
Sobald eine M3U8-URL gefunden wurde, erscheint ein Download-Icon (↓) im Player-Overlay. Tap → Quality-Auswahl Dialog (480p / 720p / 1080p — falls Varianten existieren).
3
HLS-Download via Media3 DownloadService
media3-exoplayer-hls + media3-exoplayer-workmanager: Lädt alle .ts Segmente herunter und cached sie lokal. Läuft als Foreground Service mit Notification + Fortschrittsbalken.
4
Offline abspielen
Heruntergeladene Filme werden aus dem lokalen Cache abgespielt — kein Internet nötig. Im Downloads-Tab werden alle offline verfügbaren Filme angezeigt, mit Dateigröße und Lösch-Option.
5
Storage Management
Eigenes Settings-Menü zeigt wie viel Speicher alle Downloads belegen. Einzelne Downloads löschen oder alle auf einmal clearen. Warnung bei <500MB freiem Speicher.
PlayerFragment.kt — M3U8 URL interceptenoverride fun shouldInterceptRequest(
    view: WebView,
    request: WebResourceRequest
): WebResourceResponse? {
    val url = request.url.toString()
    // M3U8 URL gefunden → an ViewModel übergeben
    if (url.contains(".m3u8") || url.contains("m3u8")) {
        requireActivity().runOnUiThread {
            viewModel.onM3u8Found(url)   // Download-Button anzeigen
        }
    }
    // Whitelist-Check (wie vorher)
    val host = request.url.host?.lowercase() ?: return super.shouldInterceptRequest(view, request)
    val allowed = setOf("nepu.to", "vr-m.net")
    if (!allowed.any { host == it || host.endsWith(".$it") }) {
        return WebResourceResponse("text/plain", "utf-8", null)  // blockiert
    }
    return null
}

// Download starten
fun startHlsDownload(m3u8Url: String, filmTitle: String) {
    val mediaItem = MediaItem.fromUri(m3u8Url)
    val request = DownloadRequest.Builder(filmTitle, mediaItem.localConfiguration!!.uri)
        .setMimeType(MimeTypes.APPLICATION_M3U8)
        .build()
    DownloadService.sendAddDownload(context, NepuDownloadService::class.java, request, false)
}
